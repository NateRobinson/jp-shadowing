# 第11課 私はマリー・マルタンです。

> 教材：文化初級日本語 I（テキスト 改訂版）
> ページ：127
> 音声：1-2-08（1-2-08.mp3）

## 本文 1

**（自己紹介）**

**マリー：** 私はマリー・マルタンです。カナダのトロントから来ました。

**マリー：** 今、横浜に住んでいます。

**マリー：** 私はカナダの高校を卒業してからファッション関係の会社に勤めていました。

**マリー：** 今年の３月に会社を辞めて、日本へファッションの勉強をしに来ました。

**マリー：** 私は自分でデザインして洋服を作るのが好きです。

**マリー：** いろいろなデザイナーの服を見るのも好きで、ときどき友達とファッションショーに行きます。

**マリー：** 私は日本の着物にとても興味があります。

**マリー：** どうぞよろしくお願いします。

## 中文翻译

**（自我介绍）**

**玛丽：** 我是玛丽·马尔坦。我从加拿大的多伦多来。

**玛丽：** 现在住在横滨。

**玛丽：** 我在加拿大高中毕业后，一直在时尚相关的公司工作。

**玛丽：** 今年3月辞了职，来日本学习时尚。

**玛丽：** 我喜欢自己设计并制作衣服。

**玛丽：** 也喜欢看各种设计师的服装，偶尔和朋友一起去看时装秀。

**玛丽：** 我对日本的和服非常感兴趣。

**玛丽：** 请多多关照。

## 重点句型

| 句型 | 含义 |
|------|------|
| ～から来ました | 从～来的 |
| ～に住んでいます | 住在～ |
| ～を卒業してから～に勤めていました | ～毕业后在～工作 |
| ～を辞めて、～しに来ました | 辞了～，来做～ |
| ～のが好きです | 喜欢做～ |
| ～にとても興味があります | 对～非常感兴趣 |
