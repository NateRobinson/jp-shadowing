# 第15課 どれがいちばん静かですか。

> 教材：文化初級日本語 I（テキスト 改訂版）
> ページ：171
> 音声：1-2-39（1-2-39.mp3）

## 本文 3

**（不動産屋で）**

**不動産屋：** じゃあ、これはどうですか。池袋駅から歩いて4分です。6畳の洋室と6畳のダイニングキッチンです。使いやすいと思います。

**吉田：** いくらですか。

**不動産屋：** 7万円です。でも、ちょっと古いです。

**吉田：** そうですか。あのう、中野と三鷹と池袋の部屋の中で、どれがいちばん静かですか。

**不動産屋：** 中野の部屋がいちばん静かだと思います。

**吉田：** そうですか、それにします。

## 中文翻译

**（在房产中介）**

**房产中介：** 那这间怎么样？从池袋车站走路4分钟。有6叠的西式房间和6叠的餐厅厨房。我觉得很方便实用。

**吉田：** 多少钱？

**房产中介：** 7万日元。不过有点旧。

**吉田：** 是吗。请问，中野、三鹰和池袋的房间里面，哪间最安静？

**房产中介：** 我觉得中野的房间最安静。

**吉田：** 是吗，那就选那间吧。

## 重点句型

| 句型 | 含义 |
|------|------|
| ～から歩いて～分です | 从～走路～分钟 |
| ～と思います | 我觉得～ |
| ～の中で、どれがいちばん～ですか | 在～当中，哪个最～ |
| ～がいちばん～だと思います | 我觉得～最～ |
| それにします | 就选那个 |
