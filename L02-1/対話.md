# 第2課 吉田良子さんの一日

> 教材：文化初級日本語 I（テキスト 改訂版）
> ページ：32
> 音声：1-1-32（1-1-32.mp3）

## 本文 1

**（吉田良子さんの一日）**

**午前 7:00：** 7時半に起きます。コーヒーを飲みます。

**午前 8:00：** 学校へ行きます。

**午前 9:00：** 9時から3時まで学校で勉強をします。

**午後 3:00：** 3時半からテニスをします。

**午後 6:00：** 6時にうちへ帰ります。晩ごはんを食べます。

**午後 7:00：** テレビを見ます。

**午後 11:00：** 11時半に寝ます。

## 中文翻译

**（吉田良子的一天）**

**上午 7:00：** 七点半起床。喝咖啡。

**上午 8:00：** 去学校。

**上午 9:00：** 从9点到3点在学校学习。

**下午 3:00：** 从三点半开始打网球。

**下午 6:00：** 6点回家。吃晚饭。

**下午 7:00：** 看电视。

**下午 11:00：** 十一点半睡觉。

## 重点句型

| 句型 | 含义 |
|------|------|
| ～時に～ます | 在～点做～（表示动作发生的时间） |
| ～へ行きます / 帰ります | 去～ / 回～（方向助词 へ） |
| ～から～まで | 从～到～（表示时间范围） |
| ～で～をします | 在～做～（场所助词 で） |
| ～を飲みます / 食べます / 見ます | 喝～ / 吃～ / 看～（宾语助词 を） |
