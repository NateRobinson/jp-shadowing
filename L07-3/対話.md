# 第7課 ラフルさんの財布じゃありませんか。

> 教材：文化初級日本語 I（テキスト 改訂版）
> ページ：93
> 音声：1-1-71（1-1-71.mp3）

## 本文 3

**（学生会館で）**

**ラフル：** ただいま。

**会館の先生：** あ、ラフルさん、おかえりなさい。財布、ありましたか。

**ラフル：** いいえ…。

**会館の先生：** ラフルさん、お金はだいじょうぶですか。

**ラフル：** はい、チンさんに借ります。

**会館の先生：** そうですか。私も貸しますよ。

**ラフル：** ありがとうございます。

**（ラフルさんの部屋の前で）**

**チン：** あ、ラフルさん。これ、ラフルさんの財布じゃありませんか。

**ラフル：** あっ、そうです。どこにありましたか。

**チン：** ラフルさんの部屋の前にありました。

**ラフル：** ああ、そうですか。どうもありがとうございました。

## 中文翻译

**（在学生会馆）**

**拉弗尔：** 我回来了。

**会馆老师：** 啊，拉弗尔，你回来了。钱包找到了吗？

**拉弗尔：** 没有……

**会馆老师：** 拉弗尔，钱还够用吗？

**拉弗尔：** 够的，我跟小陈借。

**会馆老师：** 是吗。我也可以借给你。

**拉弗尔：** 谢谢。

**（在拉弗尔房间门前）**

**小陈：** 啊，拉弗尔。这个，不是拉弗尔的钱包吗？

**拉弗尔：** 啊，是的。在哪里找到的？

**小陈：** 在拉弗尔房间门前。

**拉弗尔：** 啊，是吗。非常感谢。

## 重点句型

| 句型 | 含义 |
|------|------|
| ～じゃありませんか | 不是～吗？ |
| ～に借ります | 向～借 |
| ～も貸しますよ | 也借给（你） |
| おかえりなさい | 你回来了（欢迎回来） |
| だいじょうぶですか | 没问题吗？/还好吗？ |
| ～の前にありました | 在～前面 |
| どうもありがとうございました | 非常感谢（过去式，表示已完成的帮助） |
