# 第4課 あの大きいかばんは誰のですか。

## 本文 2

**（校外学習で）**

**先生：** このかばんは誰のですか。

**チン：** それはマリーさんのです。

**先生：** チンさんのかばんはどれですか。

**チン：** その黒いのです。

**先生：** あの大きいかばんは誰のですか。

**ワン：** あれはラフルさんのです。

## 中文翻译

**（校外学习时）**

**老师：** 这个包是谁的？

**陈：** 那是玛丽同学的。

**老师：** 陈同学的包是哪个？

**陈：** 是那个黑色的。

**老师：** 那个大包是谁的？

**王：** 那是拉弗尔同学的。

## 重点句型

| 句型 | 含义 |
|------|------|
| この / その / あの + 名词 | 这个～ / 那个～ / 那个～（指示连体词） |
| ～は誰のですか | ～是谁的？ |
| ～はどれですか | ～是哪个？ |
| その黒いのです | 是那个黑色的（の 代替名词） |
| あれは～のです | 那个是～的 |
