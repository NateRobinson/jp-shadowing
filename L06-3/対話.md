# 第6課 サッカーを見るのが好きです。

## 本文 3

**（鈴木さんのうちで）**

**吉田：** 佐藤さんはよくスポーツをしますか。

**佐藤：** いいえ、あまりしません。吉田さんは？

**吉田：** 私はよくテニスをします。サッカーも好きです。

**佐藤：** サッカーをしますか。

**吉田：** いいえ、しません。私はサッカーを見るのが好きです。佐藤さんもよくサッカーを見ますか。

**佐藤：** いいえ、あまり見ません。私はすもうを見るのが好きです。吉田さんはすもうを見ますか。

**吉田：** いいえ、ぜんぜん見ません。

## 中文翻译

**（在铃木家里）**

**吉田：** 佐藤先生经常做运动吗？

**佐藤：** 不，不怎么做。吉田小姐呢？

**吉田：** 我经常打网球。也喜欢足球。

**佐藤：** 踢足球吗？

**吉田：** 不，不踢。我喜欢看足球。佐藤先生也经常看足球吗？

**佐藤：** 不，不怎么看。我喜欢看相扑。吉田小姐看相扑吗？

**吉田：** 不，完全不看。

## 重点句型

| 句型 | 含义 |
|------|------|
| ～を見るのが好きです | 喜欢看～（动词原形+のが好き） |
| ～も好きです | 也喜欢～ |
| ～さんは？ | ～呢？（省略句尾的疑问） |
