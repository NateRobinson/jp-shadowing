# JP Shadowing - 日语影子跟读

基于《文化初級日本語 I》的影子跟读（シャドーイング）练习工具。

37 个对话场景，覆盖第1課到第15課，内置音频播放器，打开即用。

**在线使用：** https://www.myvibe.so/xwgu007/jp-shadow-reading-1

## 页面功能

### 课程导航

- **侧边栏** — 按课分组，37 个对话一键切换
- **前后切换** — 顶部 ← → 按钮，或 Alt+← / Alt+→ 快捷键
- **记忆进度** — 自动恢复上次学习的对话

### 音频播放器

- **播放/暂停** — 点击播放按钮或空格键
- **进度拖拽** — 点击或拖动进度条跳转
- **速度调节** — 0.5x / 0.75x / 1x / 1.25x / 1.5x
- **循环播放 (LOOP)** — 全文无限循环
- **A-B 复读** — 设定区间循环，攻克难句
- **播放计数** — 自动记录累计播放次数

### 内容切换

- **振り仮名** — 显示/隐藏假名注音
- **中文** — 显示/隐藏中文翻译

### 其他

- **明暗主题** — Dark / Light 模式切换
- **移动端适配** — 手机上完美显示
- **句子同步高亮** — 播放时自动高亮当前句子，点击句子跳转播放

## 练习方法

1. **第一遍** — 开启振り仮名和中文，0.75x 慢速播放，熟悉内容
2. **第二遍** — 关闭中文，1x 正常速度，跟着音频同步朗读（影子跟读）
3. **第三遍** — 关闭振り仮名，专注听力和汉字认读
4. **难句攻克** — 用 A-B 复读锁定不熟练的句子，反复跟读直到流畅
