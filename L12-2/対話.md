# 第12課 たまねぎは柔らかくなりましたか。

> 教材：文化初級日本語 I（テキスト 改訂版）

> 音声：1-2-18（1-2-18.mp3）

## 本文 2

**（料理教室で）**

**先生：** では、始めます。まず、牛肉と野菜を切ってください。

＊

**先生：** みなさん、フライパンに調味料を入れて、火をつけましたか。

**ワン：** はい、つけました。

**先生：** じゃあ、そこにたまねぎを入れて、5分ぐらい煮てください。

＊

**先生：** みなさん、たまねぎは柔らかくなりましたか。

**ワン：** はい、柔らかくなりました。

**先生：** では、次に肉を入れて、2、3分煮てください。

**アルン：** 先生、もういいですか。

**先生：** はい、じゃ、火を止めてください。

**アルン：** 早くて簡単ですね。

**先生：** ええ、薄切りの牛肉は長く煮ると、固くなります。注意してください。

**アルン：** はい。

**先生：** じゃあ、温かいごはんの上にのせて、その上にねぎをのせてください。

**アルン：** このぐらいでいいですか。

**先生：** はい、いいですよ。これでできあがりです。

**ワン：** わあ、できましたね！

＊

**ワン・アルン：** いただきます。

**ワン：** おいしい！

## 中文翻译

**（在烹饪教室）**

**老师：** 那么，开始吧。首先，请把牛肉和蔬菜切好。

＊

**老师：** 大家，调味料放进平底锅里，火点上了吗？

**王：** 是的，点上了。

**老师：** 那么，把洋葱放进去，煮大约5分钟。

＊

**老师：** 大家，洋葱变软了吗？

**王：** 是的，变软了。

**老师：** 那么，接下来把肉放进去，煮2、3分钟。

**阿伦：** 老师，可以了吗？

**老师：** 是的，请把火关掉。

**阿伦：** 又快又简单呢。

**老师：** 是的，薄切的牛肉如果煮太久的话，会变硬。请注意。

**阿伦：** 好的。

**老师：** 那么，盛到热米饭上面，再在上面放上葱。

**阿伦：** 这么多可以吗？

**老师：** 可以的。这样就做好了。

**王：** 哇，做好了呢！

＊

**王·阿伦：** 我开动了。

**王：** 好好吃！

## 重点句型

| 句型 | 含义 |
|------|------|
| ～を切ってください | 请切～ |
| ～に～を入れて、火をつけましたか | 在～里放入～，火点上了吗 |
| ～ぐらい煮てください | 请煮大约～ |
| ～は柔らかくなりましたか | ～变软了吗 |
| ～く煮ると、～くなります | 如果煮得～，就会变～ |
| ～の上にのせて | 放在～上面 |
| このぐらいでいいですか | 这么多可以吗 |
| これでできあがりです | 这样就做好了 |
