# 第9課 24日までに返してください。

> 教材：文化初級日本語 I（テキスト 改訂版）
> ページ：117
> 音声：1-1-84（1-1-84.mp3）

## 本文 3

**（図書館のカウンターで）**

**ワン：** この本をお願いします。

**図書館員：** はい。24日までに返してください。

**ワン：** はい、わかりました。

**（図書館の前で）**

**ワン：** ラフルさんは何か借りましたか。

**ラフル：** いいえ、何も借りませんでした。ワンさんは何か借りましたか。

**ワン：** はい、日本の着物の本を借りました。これです。

**ラフル：** ああ、きれいな着物ですね。

**ワン：** そうですね。

## 中文翻译

**（在图书馆柜台）**

**王：** 我要借这本书。

**图书馆员：** 好的。请在24号之前归还。

**王：** 好的，我知道了。

**（在图书馆前面）**

**王：** 拉弗尔，你借了什么吗？

**拉弗尔：** 没有，什么都没借。王同学你借了什么？

**王：** 是的，我借了一本关于日本和服的书。就是这本。

**拉弗尔：** 哇，好漂亮的和服啊。

**王：** 是啊。

## 重点句型

| 句型 | 含义 |
|------|------|
| ～までに返してください | 请在～之前归还 |
| ～は何か借りましたか | ～借了什么吗？ |
| 何も～ませんでした | 什么都没～ |
| ～の本を借りました | 借了～的书 |
| きれいな～ですね | 好漂亮的～啊 |
