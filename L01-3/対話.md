# 第1課 休みはいつですか。

> 教材：文化初級日本語 I（テキスト 改訂版）
> ページ：26
> 音声：1-1-28（1-1-28.mp3）

## 本文 3

**（教室で）**

**先生：** 授業は9時10分からです。

**ワン：** 何時までですか。

**先生：** 2時50分までです。

**リー：** 昼休みは何時から何時までですか。

**先生：** 12時から1時までです。

**ラフル：** あのう、休みはいつですか。

**先生：** 土曜日と日曜日です。

## 中文翻译

**（在教室里）**

**老师：** 上课从9点10分开始。

**王：** 到几点结束？

**老师：** 到2点50分。

**李：** 午休是从几点到几点？

**老师：** 从12点到1点。

**拉弗尔：** 请问，休息日是什么时候？

**老师：** 星期六和星期天。

## 重点句型

| 句型 | 含义 |
|------|------|
| ～は～からです。 | ～从～开始。（表示起点） |
| ～までですか。 | 到～吗？（表示终点） |
| ～から～まで | 从～到～（时间范围） |
| ～はいつですか。 | ～是什么时候？（询问时间） |
| ～と～ | ～和～（并列连接） |
