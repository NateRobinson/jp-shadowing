# 第11課 デザインの勉強をしたいです。

> 教材：文化初級日本語 I（テキスト 改訂版）

> 音声：1-2-12（1-2-12.mp3）

## 本文 3

**（先生とマリーの会話）**

**先生：** マリーさんはこの学校を卒業してからどうしますか。

**マリー：** デザインの勉強をしたいです。

**マリー：** 将来、デザイナーになりたいです。

**先生：** そうですか。もう学校を決めましたか。

**マリー：** いいえ、まだはっきり決めていません。

**マリー：** でも、専門学校に入りたいです。

**先生：** もうご家族と相談しましたか。

**マリー：** いいえ、まだです。

**マリー：** もう少し調べてから相談します。

## 中文翻译

**（老师和玛丽的对话）**

**老师：** 玛丽从这所学校毕业后打算做什么？

**玛丽：** 我想学习设计。

**玛丽：** 将来想成为设计师。

**老师：** 是吗。已经决定好学校了吗？

**玛丽：** 不，还没有明确决定。

**玛丽：** 但是，我想进专门学校。

**老师：** 已经和家人商量过了吗？

**玛丽：** 不，还没有。

**玛丽：** 想再查一查之后再商量。

## 重点句型

| 句型 | 含义 |
|------|------|
| ～を卒業してからどうしますか | ～毕业后打算怎么办 |
| ～たいです | 想要做～ |
| ～になりたいです | 想要成为～ |
| もう～ましたか | 已经～了吗 |
| まだ～ていません | 还没有～ |
| ～に入りたいです | 想进入～ |
| ～てから～します | 做了～之后再～ |
