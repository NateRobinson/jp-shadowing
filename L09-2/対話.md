# 第9課 本の探し方を説明します。

> 教材：文化初級日本語 I（テキスト 改訂版）
> ページ：111
> 音声：1-1-81（1-1-81.mp3）

## 本文 2

**（図書館で）**

**図書館員：** これから本の探し方を説明します。よく聞いてください。

**学生：** はい。

**図書館員：** まず、ここに本の名前を入れて、ここを押してください。

**ラフル：** すみません、どこを押しますか。

**図書館員：** ここです。

**ラフル：** ああ、わかりました。ありがとうございます。

**（本棚の前で）**

**キム：** ワンさん、本を取りましょうか。

**ワン：** すみません。お願いします。

**キム：** はい、どうぞ。

**ワン：** ありがとうございます。

## 中文翻译

**（在图书馆）**

**图书馆员：** 现在我来说明查找书籍的方法。请认真听。

**学生：** 好的。

**图书馆员：** 首先，在这里输入书名，然后按这里。

**拉弗尔：** 不好意思，按哪里？

**图书馆员：** 这里。

**拉弗尔：** 啊，明白了。谢谢。

**（在书架前）**

**金：** 王同学，我帮你拿书吧？

**王：** 不好意思，麻烦你了。

**金：** 给你。

**王：** 谢谢。

## 重点句型

| 句型 | 含义 |
|------|------|
| ～の探し方を説明します | 说明～的查找方法 |
| ～を入れて、～を押してください | 请输入～，然后按～ |
| どこを押しますか | 按哪里？ |
| ～を取りましょうか | 我帮你拿～吧？ |
| お願いします | 麻烦你了 |
