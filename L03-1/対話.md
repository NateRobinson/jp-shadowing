# 第3課 これは誰のノートですか。

> 教材：文化初級日本語 I（テキスト 改訂版）
> ページ：42
> 音声：1-1-39（1-1-39.mp3）

## 本文 1

**（教室で）**

**先生：** これは誰のノートですか。

**ラフル：** それは私のノートです。

**先生：** これもラフルさんのですか。

**ラフル：** いいえ、それは私のじゃありません。

## 中文翻译

**（在教室里）**

**老师：** 这是谁的笔记本？

**拉弗尔：** 那是我的笔记本。

**老师：** 这也是拉弗尔同学的吗？

**拉弗尔：** 不，那不是我的。

## 重点句型

| 句型 | 含义 |
|------|------|
| これ / それ | 这个 / 那个（指示代词） |
| ～は誰のですか | ～是谁的？（所有者疑问） |
| ～も～ですか | ～也是～吗？（助词 も 表示"也"） |
| ～のじゃありません | 不是～的（所有格否定） |
