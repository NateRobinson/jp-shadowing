# 第8課 天気はどうでしたか。

> 教材：文化初級日本語 I（テキスト 改訂版）
> ページ：95
> 音声：1-1-73（1-1-73.mp3）

## 本文 1

**ニュースキャスター：** こんばんは。今日は5月5日、子供の日です。今年のゴールデンウィークは今日で終わりました。今日、約5万人の人が帰国しました。成田空港から伊藤記者が報告します。

**（成田空港で）**

**伊藤記者：** あのう、すみません。

**女の人：** はい。

**伊藤記者：** どこへいらっしゃいましたか。

**女の人：** ハワイへ行きました。

**伊藤記者：** 何日間ですか。

**女の人：** 1週間です。

**伊藤記者：** 旅行はどうでしたか。

**女の人：** 少し疲れましたが、楽しかったです。

**伊藤記者：** 天気はどうでしたか。

**女の人：** 毎日とてもいい天気でした。でも、ちょっと暑かったですけど。朝と夜はあまり暑くありませんでした。

**伊藤記者：** いつごろ切符を予約しましたか。

**女の人：** 3か月前です。

**伊藤記者：** 予約は大変でしたか。

**女の人：** いいえ、あまり大変じゃありませんでした。

## 中文翻译

**新闻主播：** 晚上好。今天是5月5日，儿童节。今年的黄金周今天就结束了。今天约有5万人回国了。下面由成田机场的伊藤记者进行报道。

**（在成田机场）**

**伊藤记者：** 那个，不好意思。

**女士：** 嗯。

**伊藤记者：** 您去了哪里？

**女士：** 去了夏威夷。

**伊藤记者：** 去了几天？

**女士：** 一周。

**伊藤记者：** 旅行怎么样？

**女士：** 虽然有点累，但是很开心。

**伊藤记者：** 天气怎么样？

**女士：** 每天都是非常好的天气。不过有点热。早上和晚上不太热。

**伊藤记者：** 什么时候预约的机票？

**女士：** 三个月前。

**伊藤记者：** 预约困难吗？

**女士：** 不，不太难。

## 重点句型

| 句型 | 含义 |
|------|------|
| ～はどうでしたか | ～怎么样？（过去） |
| ～で終わりました | 到～就结束了 |
| ～へいらっしゃいましたか | 您去了～？（敬语） |
| 何日間ですか | 几天？ |
| ～ましたが、～かったです | 虽然～，但是～ |
| ～くありませんでした | 不～（い形容词过去否定） |
| いつごろ～ましたか | 什么时候～了？ |
| あまり～じゃありませんでした | 不太～（な形容词过去否定） |
