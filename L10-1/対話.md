# 第10課 東京から熱海まで、新幹線で行きます。

> 教材：文化初級日本語 I（テキスト 改訂版）
> ページ：120
> 音声：1-2-02（1-2-02.mp3）

## 本文 1

**（東京駅で）**

**ガイド：** みなさん、今から今日の予定を言います。

**ガイド：** まず、東京から熱海まで、新幹線こだま６０３号で行きます。

**ガイド：** こだま６０３号は、午後２時２０分に東京駅を出て、３時１７分に熱海に着きます。

**ガイド：** 熱海で新幹線を降りて、バスに乗りかえて、下田までバスで行きます。

**ガイド：** 途中、伊東でバスを降りて休憩します。

**ガイド：** 熱海から下田まで２時間ぐらいかかります。

**ガイド：** ５時半ごろ下田に着きます。

## 中文翻译

**（在东京站）**

**导游：** 大家好，现在我来说一下今天的行程安排。

**导游：** 首先，从东京到热海，我们乘坐新干线回声号603次列车。

**导游：** 回声号603次列车下午2点20分从东京站出发，3点17分到达热海。

**导游：** 在热海下新干线，换乘巴士，坐巴士前往下田。

**导游：** 途中在伊东下车休息。

**导游：** 从热海到下田大约需要2个小时。

**导游：** 5点半左右到达下田。

## 重点句型

| 句型 | 含义 |
|------|------|
| ～から～まで、～で行きます | 从～到～，乘坐～去 |
| ～時～分に～を出て | ～点～分从～出发 |
| ～に着きます | 到达～ |
| ～を降りて、～に乗りかえて | 下～，换乘～ |
| ～時間ぐらいかかります | 大约需要～小时 |
| ～ごろ～に着きます | ～左右到达～ |
