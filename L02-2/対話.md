# 第2課 昨日、何をしましたか。

> 教材：文化初級日本語 I（テキスト 改訂版）
> ページ：39
> 音声：1-1-34（1-1-34.mp3）

## 本文 2

**（月曜日　教室で）**

**ラフル：** おはようございます。

**ワン：** おはようございます。

**ラフル：** ワンさん、昨日、何をしましたか。

**ワン：** 渋谷で映画を見ました。ラフルさんは？

**ラフル：** 私はテニスをしました。

**ワン：** そうですか。

## 中文翻译

**（周一　在教室里）**

**拉弗尔：** 早上好。

**王：** 早上好。

**拉弗尔：** 王同学，昨天做了什么？

**王：** 在涩谷看了电影。拉弗尔同学呢？

**拉弗尔：** 我打了网球。

**王：** 是吗。

## 重点句型

| 句型 | 含义 |
|------|------|
| ～を～ました | 做了～（动词过去时） |
| ～で～を見ました | 在～看了～（场所助词 で + 过去时） |
| ～さんは？ | ～呢？（用は提示话题，省略谓语） |
| 何をしましたか | 做了什么？（疑问词 何 + 过去时） |
| そうですか | 是吗。（表示理解/回应） |
