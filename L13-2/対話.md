# 第13課 どう思いますか。

> 教材：文化初級日本語 I（テキスト 改訂版）
> ページ：153
> 音声：1-2-24（1-2-24.mp3）

## 本文 2

**ラフル：** 暇な時は、何をしますか。

**田中：** よく友達とショッピングに行きます。

**加藤：** 私はスポーツをします。

**ラフル：** どんなスポーツをしますか。

**加藤：** バスケットボールです。

**ラフル：** そうですか。ところで、この学校の施設をどう思いますか。

**田中：** 図書館が新しいから、いいと思います。

**加藤：** 食堂は安くておいしいから、よく行きます。

**田中：** 食堂のソフトクリームはとてもおいしいですよ。

## 中文翻译

**拉弗尔：** 空闲的时候做什么？

**田中：** 经常和朋友去购物。

**加藤：** 我做运动。

**拉弗尔：** 做什么运动？

**加藤：** 篮球。

**拉弗尔：** 是吗。对了，你们觉得这所学校的设施怎么样？

**田中：** 图书馆很新，我觉得很好。

**加藤：** 食堂又便宜又好吃，我经常去。

**田中：** 食堂的软冰淇淋也非常好吃哦。

## 重点句型

| 句型 | 含义 |
|------|------|
| ～な時は、何をしますか | 在～的时候做什么 |
| よく～に行きます | 经常去～ |
| どんな～をしますか | 做什么样的～ |
| ～をどう思いますか | 你觉得～怎么样 |
| ～が～から、いいと思います | 因为～，我觉得很好 |
| ～は～くて～から、よく～ます | 因为～又～又～，所以经常～ |
