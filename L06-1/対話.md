# 第6課 料理をしますか。

> 教材：文化初級日本語 I（テキスト 改訂版）
> ページ：70-71
> 音声：1-1-58（1-1-58.MP3）

## 本文 1

**（鈴木さんのうちで）**

**鈴木幸子：** みなさん、サンドイッチ、どうぞ。

**吉田良子：** ありがとうございます。

**みんな：** いただきます。おいしい！

**幸子：** 武さん、こちら、私の友達の吉田良子さんです。

**吉田：** はじめまして。吉田です。

**佐藤武：** 佐藤武です。よろしく。

**佐藤：** 幸子さんの料理はおいしいですね。吉田さんは料理をしますか。

**吉田：** いいえ、あまりしません。

**佐藤：** そうですか。

**吉田：** 佐藤さんはよくお酒を飲みますか。

**佐藤：** いいえ、ぜんぜん飲みません。吉田さんはよく飲みますか。

**吉田：** ときどき飲みます。

**佐藤：** 何を飲みますか。

**吉田：** ビールかワインを飲みます。

## 中文翻译

**（在铃木家里）**

**铃木幸子：** 大家，三明治，请吃。

**吉田良子：** 谢谢。

**大家：** 我开动了。好好吃！

**幸子：** 武先生，这位是我的朋友吉田良子。

**吉田：** 初次见面。我是吉田。

**佐藤武：** 我是佐藤武。请多关照。

**佐藤：** 幸子做的菜真好吃啊。吉田小姐做菜吗？

**吉田：** 不，不怎么做。

**佐藤：** 是吗。

**吉田：** 佐藤先生经常喝酒吗？

**佐藤：** 不，完全不喝。吉田小姐经常喝吗？

**吉田：** 偶尔喝。

**佐藤：** 喝什么？

**吉田：** 喝啤酒或红酒。

## 重点句型

| 句型 | 含义 |
|------|------|
| こちら、～さんです | 这位是～（介绍他人） |
| ～をしますか | 做～吗？ |
| あまり～ません | 不怎么～（部分否定） |
| ぜんぜん～ません | 完全不～（全面否定） |
| よく～ますか | 经常～吗？ |
| ときどき～ます | 偶尔～ |
| ～か～を～ます | ～或～（选择） |
