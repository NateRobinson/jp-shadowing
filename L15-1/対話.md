# 第15課 どちらのほうが近いですか。

> 教材：文化初級日本語 I（テキスト 改訂版）
> ページ：166
> 音声：1-2-33（1-2-33.mp3）

## 本文 1

**（不動産屋で）**

**不動産屋：** このアパートは中野と新中野の間です。

**吉田良子：** 中野と新中野とどちらのほうが近いですか。

**不動産屋：** 中野のほうがちょっと近いです。でも、どちらもだいたい同じぐらいですよ。

**吉田：** 中野の駅前と新中野の駅前とどちらのほうがにぎやかですか。

**不動産屋：** 中野のほうがずっとにぎやかです。北口に大きい商店街がありますよ。

**吉田：** そうですか。家賃はいくらですか。

**不動産屋：** 8万円です。

**吉田：** ちょっと高いですね。

## 中文翻译

**（在房产中介）**

**房产中介：** 这间公寓在中野和新中野之间。

**吉田良子：** 中野和新中野哪边比较近？

**房产中介：** 中野那边稍微近一点。不过两边差不多一样远。

**吉田：** 中野站前和新中野站前，哪边比较热闹？

**房产中介：** 中野那边热闹得多。北口有很大的商店街哦。

**吉田：** 是吗。房租多少？

**房产中介：** 8万日元。

**吉田：** 有点贵呢。

## 重点句型

| 句型 | 含义 |
|------|------|
| ～と～の間です | 在～和～之间 |
| ～と～とどちらのほうが～ですか | ～和～哪个比较～ |
| ～のほうが～です | ～那边比较～ |
| どちらもだいたい同じぐらいです | 两边差不多一样 |
| ～にぎやかですか | ～热闹吗 |
| ～がずっと～です | ～要～得多 |
| 家賃はいくらですか | 房租多少 |
